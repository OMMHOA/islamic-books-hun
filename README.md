# islamic-books-hun

directory with the source scans, transcriptions, and the Hungarian translation.

   - [Arabic transcription](fiqh-us-seerah/FiqhusSeerah-Muhammad-al-Ghazali-AR-full.md) (Maṭābiʿ al-Shurūq edition)
